# プロジェクト

ある期限の中でビジネス上の目的を達成するために、時間・人を含めた資源を計画すること。

ここでの取り組みでは、ロボコンに向けてメンバーの役割分担や時間管理をしているので、プロジェクトに相当する。

# プロジェクトを進める難しさ

プロジェクトには期限がある一方で、それを取り巻く不確定要素は多い。

そういった偶然やメンバーのスキルに左右されることなく、プロジェクトを安定して成功させたい。

## チケット駆動開発（Ticket-Driven Development）

チケット駆動開発は、プロジェクトを安定して成功させることを目的として確立された仕事の進め方である。

チケット駆動開発では、"チケット"と呼ばれる作業単位を電子上で管理する。
